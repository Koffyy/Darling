`k?avatar` **Gives you someone's avatar** __Usage__: *avatar [@user]*
`k?icon` **Gives you the server icon** __Usage__: *icon*
`k?botinfo` **Gives the bot info** __Usage__: *botinfo*
`k?userinfo` **Gives you the user info** __Usage__: *userinfo [@user]*
`k?serverinfo` **Gives you the server info** __Usage__: *serverinfo*
`k?finduser`  **Finds users with this name** __Usage__: *finduser <name>*
